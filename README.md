## 🚀 How to Run the Singleton Pattern

To compile and execute the Singleton pattern test, open your terminal at the **root directory** of this project and run the following commands:

### 1. Compile
Compile both files together so the package dependencies resolve correctly:
```bash
javac patterns/singleton/BoardDirector.java patterns/singleton/SingletonTest.java
```

### 2. Run
Execute the program using its fully qualified class name:
```bash
java patterns.singleton.SingletonTest
```
